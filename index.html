Portfolio Websites

weather

This website fetches data from "openweather.com" and displays the information. There is a search bar for users to retrieve the weather information of the location entered. Switching to "detail"tab can view more information. 5 days weather is displayed by the carousel of weather cards and the location can be viewed in the google map.

The theme changes based on the weather displayed in the main view. Click on different weather card can change the main view. Also, the temperature in the main view can be toggled between degree celsius and fahrenheit. It is implemented with the responsive design.
